# SonicStream

A cinematic, futuristic music streaming platform. Your personal music universe without boundaries.

Made with ❤️ by **Pratik Soni**

## 🚀 Features

- **Multi-Source Sync**: Paste URLs from Spotify, YouTube, SoundCloud, and more.
- **Cinematic Experience**: Immersive dark UI with glassmorphism and beat-reactive visuals.
- **Seamless Playback**: High-performance audio extraction and gapless transitions.
- **AI-Driven Logic**: Smart metadata extraction and mood-based recommendations.
- **Cross-Platform Readiness**: Designed for web, mobile, and desktop.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, TailwindCSS, Motion, Zustand.
- **Backend**: Express (Node.js), Gemini AI (Advanced metadata extraction).
- **Database/Auth**: Firebase Firestore & Authentication.

## 📖 GitHub Help Guide

To deploy this project or contribute via GitHub, follow these steps:

### 1. Cloning the Repository
```bash
git clone https://github.com/your-username/sonicstream.git
cd sonicstream
```

### 2. Environment Setup
Create a `.env` file in the root directory and add your API keys:
```env
GEMINI_API_KEY=your_key_here
FIREBASE_CONFIG=your_json_config
```

### 3. Installation
```bash
npm install
```

### 4. Running Development Server
```bash
npm run dev
```

### 5. Building for Production
```bash
npm run build
```

The production output will be in the `dist/` directory.

### 6. Deployment
You can deploy the `dist/` folder to platforms like Vercel, Netlify, or Firebase Hosting. The backend should be deployed to a Node.js compatible environment (like Cloud Run or Heroku).
